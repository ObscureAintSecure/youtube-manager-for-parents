# Privacy Policy - YouTube Kids Manager

Last updated: June 10, 2026

YouTube Kids Manager is a Chrome extension that helps parents manage a child's YouTube subscriptions and recommendations.

## Data collection

This extension does not collect, store remotely, transmit, sell, or share any user data. Nothing leaves your device.

## What the extension accesses

- It reads channel names and links visible on the YouTube page you have open, only when you click an action in the popup.
- It clicks YouTube's own unsubscribe and "Don't recommend channel" controls on your behalf.
- Batch progress and results are kept in Chrome's local extension storage on your device so a running batch survives the popup closing. This data never leaves your browser and is cleared when you press Done.

## Third parties

No user data is sent to the developer or to any third party.

The extension makes one kind of outbound request, and only when you click "Sync from GitHub" in the Lists tab: it downloads the latest channel block lists (plain text files) from this project's public GitHub repository. That request sends no personal data; it is an ordinary file download. The extension makes no other network requests; everything else happens in the YouTube page already open in your browser.
