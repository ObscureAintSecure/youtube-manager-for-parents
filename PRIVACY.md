# Privacy Policy - YouTube Manager for Parents

Last updated: June 14, 2026

YouTube Manager for Parents is a Chrome extension that helps parents clean up a child's YouTube subscriptions, recommendations, and watch history, and subscribe to a shared list of recommended channels.

## Data collection

This extension does not collect, store remotely, transmit, sell, or share any personal data. No personal or browsing data leaves your device.

## What the extension accesses

- It reads channel names, handles, and video titles visible on the YouTube page you have open, only when you click an action in the popup.
- It clicks YouTube's own controls on your behalf: unsubscribe, "Don't recommend channel", "Remove from watch history", and Subscribe.
- Batch progress and results, and your cached channel lists, are stored in Chrome's local extension storage on your device. This data never leaves your browser.

## Network requests

The only outbound request is downloading the extension's public channel lists (plain text files) from this project's GitHub repository. This happens when the popup opens and when you click Refresh. It sends no personal data; it is an ordinary file download. If GitHub is unreachable, a locally cached copy is used.

When you choose to suggest a channel, the extension opens a prefilled GitHub issue page in a new tab; nothing is submitted unless you post it yourself.

The extension makes no other network requests. All other actions happen in the YouTube page already open in your browser, using your existing YouTube session.
